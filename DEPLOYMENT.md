# Deployment Checklist for TalentLink

## Pre-Deployment Steps
- [ ] All local testing completed
- [ ] Database migrations tested
- [ ] Frontend build successful locally
- [ ] Environment variables configured

## Single Render Project Deployment
This project uses a single Render project with multiple services:

### Services:
1. **Backend Service**: Django API
   - URL: https://talentlink-backend.onrender.com
   - Environment: Python
   - Database: PostgreSQL

2. **Frontend Service**: React Static Site
   - URL: https://talentlink-frontend.onrender.com
   - Environment: Static Site
   - Build: npm run build

### Environment Variables:
Backend:
- DATABASE_URL: Auto-configured by Render
- SECRET_KEY: Auto-generated by Render
- DEBUG: false
- ALLOWED_HOSTS: "*"

Frontend:
- REACT_APP_API_URL: https://talentlink-backend.onrender.com
- REACT_APP_ENVIRONMENT: production

## Deployment Commands
```bash
# Deploy to production (automatic via GitHub)
git add .
git commit -m "Deploy TalentLink to production"
git push origin main
```

## Post-Deployment Verification
- [ ] Backend API accessible at /api/
- [ ] Frontend loads correctly
- [ ] API calls work between frontend and backend
- [ ] Admin panel accessible at /admin/
- [ ] Database migrations applied

## Troubleshooting
- Check Render dashboard logs
- Verify environment variables
- Check database connection
- Review build logs
